# M. Sundar – Portfolio

Personal portfolio website for **M. Sundar**, a Senior IT Infrastructure & Operations Professional with 15+ years of experience in enterprise hardware, Linux administration, cloud (AWS), and team leadership.

## Tech Stack

- HTML5 + Tailwind CSS (CDN)
- Lucide Icons
- Google Fonts (Inter, Montserrat)
- Vanilla JavaScript

## Features

- Glassmorphism dark-themed design
- Animated floating navigation pill with active section indicator
- Interactive career timeline
- Responsive layout (mobile & desktop)

## Run Locally

Simply open `index.html` in any browser — no build step required.
